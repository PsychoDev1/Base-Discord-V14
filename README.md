
# Bot Discord V14

A brief description of what this project does and who it's for

##  How to install ?

```bash
  npm i discord.js
  npm i @discordjs/rest
  npm i fs
  npm i path
```

##  How to config ?

```bash
  You need to change the token in './config'
```
##  How to start ?

```bash
  node index.js # To start the bot
```
